# REST-API Project
TR -> NodeJs ve ExpressJs kullanarak route üzerinden /listele yazdığımızda 
.json uzantılı dosyamızın içerisindeki verileri getiren, /ekle yazdığımızda yapıya uygun şekilde girdiğimiz değerleri  .json uzantılı dosyaya ekleyen, 
/sil yazdığımızda girdiğimiz uygun id'ye ait veriyi .json uzantılı dosyadan silen ancak bu id'nin karşılığında veri yoksa bize uyarı mesajı atan, /sorgula yazdığımızda ise bize girdiğimiz id'ye ait veriyi .json uzantılı dosya içerisinden bulup bize ekranda gösteren bir API yazdım.

ENG -> When we write /listele over route using NodeJs and ExpressJs 
It brings the data in our .json extension file, when we type /ekle, it adds the values we enter in accordance with the structure to the .json extension file, 
When we type /sil, I wrote an API that deletes the data belonging to the appropriate id we entered from the .json extension file, but if there is no data for this id, it gives us a warning message, and when we type /sorgula, it finds the data belonging to the id we entered from the .json extension file and shows us on the screen.
